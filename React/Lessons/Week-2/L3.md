
# Installation of Vite 

## Command to create a new Vite project
-- npm create vite@latest

# Why we use Vite instead of Create React App (CRA)? ** IMPORTANT **

-- Faster Development Server: Vite uses native ES modules and a modern development server that provides instant hot module replacement (HMR). This means that changes to your code are reflected in the browser almost instantly, without the need for a full page reload. In contrast, Create React App (CRA) can be slower to start and may require a full page reload for certain changes.

-- Faster Installation and Build Times: Vite has a smaller footprint and faster installation times compared to CRA. It also uses Rollup for production builds, which can result in faster build times and smaller bundle sizes.

## Command to create a new React project using Create React App (CRA)
-- npx create-react-app my-app

Slow Installation and build times, especially for larger projects. CRA can take several minutes to install dependencies and build the application, which can be frustrating for developers.
